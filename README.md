# TrueLens

TrueLens is an AI-powered platform designed to evaluate the authenticity, quality, and reliability of real-world data.  
It detects fake, low-quality, anomalous, and drifting data using Machine Learning, Natural Language Processing, and statistical techniques.

## Key Features
- Fake / bot-generated text detection
- Data quality scoring (Trust Score)
- Anomaly and outlier detection
- Data drift monitoring over time
- Explainable AI insights
- Interactive web dashboard (Streamlit)
- Fully deployed and production-ready

## Use Cases
- Fake review detection (E-commerce, Google Maps, App Stores)
- Survey and feedback quality validation
- Social media bot content filtering
- Enterprise data quality monitoring
- Sensor and log anomaly detection (future extension)

##  Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, PyTorch / TensorFlow
- NLP (TF-IDF, Sentence Transformers)
- Isolation Forest, LOF, Autoencoders
- Streamlit / FastAPI
- Docker, GitHub Actions
- Deployed on Cloud (HuggingFace / Render)
